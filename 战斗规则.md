# 战斗系统 - 简化规则

> 本规则基于《无限恐怖TRPG》核心检定系统简化而来
> 适用于挂机放置游戏的自动战斗系统

---

## 一、核心检定

### DP（骰池）

```
DP = 属性 + 技能 + 调整值
```

DP决定投掷D10的数量。

### 成功判定

- 投出 **8, 9, 10** → 1个成功
- 投出 **1-7** → 无成功

### 加骰（Explosion）

- 投出 **10** → 再投1枚D10
- 新骰子同样遵循 8/9/10 = 成功
- 可以连锁（10 → 10 → 10...）

### 附加成功

高属性/技能提供额外成功数:

```
属性附加成功:
  属性 6-10  → +1
  属性 11-15 → +2
  属性 16-20 → +3
  属性 21+   → +4

技能附加成功:
  技能 3-5  → +1
  技能 6-8  → +2
  技能 9-11 → +3
  技能 12+  → +4
```

### 最终成功数

```
最终成功数 = 基础成功数 + 附加成功数
```

---

## 二、攻击检定

```
DP = 攻击属性 + 攻击技能
投骰 → 计算成功数A
```

### 命中判定

```
若 A > 敌人防御成功数B → 命中
若 A ≤ B → 未命中
```

### 伤害计算

```
基础伤害 = 武器伤害 + 力量/2
实际伤害 = 基础伤害 + (A - B) × 2
最终伤害 = max(1, 实际伤害 - 敌人护甲)
```

---

## 三、防御检定（自动）

```
DP = 敏捷 + 闪避技能
投骰 → 计算成功数D
```

### 闪避判定

```
若 D > 敌人攻击成功数C → 闪避成功，不受伤害
若 D ≤ C → 受到伤害 = 敌人伤害 - 防御值
```

---

## 四、战斗回合

```
回合流程:
1. 玩家攻击检定 → 投骰 → 计算成功数
2. 敌人防御检定 → 投骰 → 对比
3. 敌人攻击检定 → 投骰 → 计算成功数
4. 玩家防御检定 → 投骰 → 对比
5. 更新HP → 记录日志
6. 重复直到一方HP≤0
```

## 六、特殊规则

### 暴击

- 10加骰连锁3次以上 → 暴击
- 暴击伤害 = 正常伤害 × 1.5

### 连击

- 敏捷 ≥ 15 → 额外攻击一次
- 每10点敏捷 → +1次额外攻击

### 吸血

- 吸血比例 = 吸血值 / 100
- 回复 = 造成伤害 × 吸血比例
