## 一、项目概述

### 核心理念

- **挂机保底**: 自动获取奖励点数（无支线产出）
- **手动副本**: 核心成长路径，获取支线+XP
- **支线互换**: 3:1经典比例，高级线索稀缺

### 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| Pinia | 状态管理 |
| 原生CSS | 赛博朋克主题 |

## 三、核心系统

### 1. 三大货币系统

| 货币 | 挂机获取 | 副本获取 | 用途 |
|------|---------|---------|------|
| 奖励点数 | ✅ 每小时+50 | ✅ 大量 | 购买属性/血统/技能 |
| 支线 (D/C/B/A/S) | ❌ | ✅ 掉落 | 兑换强化能力 |
| XP | ❌ | ✅ 战斗 | 训练技能 |

**挂机收益:**
- 每小时 +500 奖励点数 (生化危机地图)
- 每小时 +100 XP (生化危机地图)
- 无支线产出
- 收益随地图等级提升

### 2. 属性系统（原著6属性）

基于《无限恐怖》原著设定，使用固定值加成避免数值堆叠：

| 属性 | 简称 | 定位 | 每点效果 | 附加成功 |
|------|------|------|---------|---------|
| 智力 | INT | 魔法资源 | MP上限+10, MP恢复+2 | 6→+1, 11→+2, 16→+3 |
| 精神力 | SPR | 魔法输出 | 精神伤害+2 | 6→+1, 11→+2, 16→+3 |
| 细胞活力 | VIT | 生存 | HP上限+20, HP恢复+1 | 6→+1, 11→+2, 16→+3 |
| 神经反应 | REA | 速度/暴击 | 闪避+1%, 暴击率+1%, 先手+2 | 6→+1, 11→+2, 16→+3 |
| 肌肉强度 | STR | 物理输出 | 物理攻击+5, 物理伤害+2 | 6→+1, 11→+2, 16→+3 |
| 免疫强度 | IMM | 防御 | 异常抗性+3%, 减伤+1 | 6→+1, 11→+2, 16→+3 |

**属性定位总结：**
- 物理输出: 肌肉强度 (攻击力+伤害)
- 魔法输出: 精神力 (魔法伤害)
- 物理防御: 免疫强度 (减伤+抗性)
- 生存能力: 细胞活力 (HP+恢复)
- 速度暴击: 神经反应 (闪避+暴击+先手)
- 魔法资源: 智力 (MP上限+恢复)

**创建规则：**
- 所有属性初始值 = 1
- 职业加成: 主属性+3, 副属性+2
- 缺陷点数兑换: 1点 → 1属性点
- 普通人上限 = 5
- 基因锁突破后上限提升

**CRPG检定系统：**

副本中遇到技能检定节点时，可选择以下检定：
- **生理检定**: 肉搏/白刃/枪械/弓术/求生
- **心智检定**: 学识/调查/医学/神秘学/科学
- **社交检定**: 感受/表达/胁迫/掩饰

检定公式:
```
DP = 属性值 + 技能等级
DC = 节点难度 (3-15)
投骰D10 → 成功数 ≥ DC → 成功
```

**DC难度表：**
| 难度 | DC | 对应副本 |
|------|-----|---------|
| 简单 | 3-5 | 丧尸围城 |
| 一般 | 6-8 | 异形巢穴 |
| 困难 | 9-11 | 猛鬼街 |

#### 物品差价升级规则

**差价升级：**
- 可以补充差价向上购买同一类型物品
- 需保证物品完好且使用次数完整
- 升级后物品提供属性上限（但不提供属性）

**差价计算：**
- 将高等级支线拆分为低等级支线
- 分数和XP直接计算加减法

**示例：**
- D+500武器 → B+2000武器
- B支线 = 9条D支线
- 差价 = 9-1 = 8条D支线
- 分数差 = 2000-500 = 1500
- 总价 = 8D+1500（或CCDD+1500）

### 7. 基因锁系统

5阶解锁，每阶大幅提升属性:
- 阶段0: 初始状态
- 阶段1-5: 逐步解锁

解锁需要消耗奖励点数 + 满足前置条件。

### 8. 传奇属性系统

当属性超过5点上限时，获得传奇属性加成。

#### 获取规则

```
传奇点数 = max(0, (属性值 - 5) / 5)

示例:
  属性5 → 0点传奇
  属性6 → 1点传奇
  属性11 → 2点传奇
  属性16 → 3点传奇
```

#### 解锁时间预估

```
挂机收入: 50奖励点/小时

主属性(+3职业): 从4开始 → 需900奖励点 → 18小时
副属性(+2职业): 从3开始 → 需1200奖励点 → 24小时
无加成属性: 从1开始 → 需1500奖励点 → 30小时
```

| 3点 | 物理伤害+24 |

## 四、战斗系统（TRPG简化版）

### 核心检定系统

```
DP（骰池）= 属性 + 技能 + 调整值
投骰: DP枚D10
成功: 每枚 ≥ 8 → 1成功
加骰: 投出10 → 再投1枚（可连锁）
附加成功: 高属性/技能额外加成

最终成功数 = 基础成功数 + 附加成功数
```

### 附加成功规则

```
属性附加成功:
  属性 6-10  → +1
  属性 11-15 → +2
  属性 16-20 → +3
  属性 21+   → +4

技能附加成功:
  技能 3-5  → +1
  技能 6-8  → +2
  技能 9-11 → +3
  技能 12+  → +4
```

### 攻击流程

```
1. 玩家攻击检定:
   近战(肉搏): DP = 肌肉强度 + 肉搏技能
   近战(白刃): DP = 神经反应 + 白刃技能
   远程(枪械): DP = 神经反应 + 枪械技能
   远程(弓术): DP = 神经反应 + 弓术技能
   投骰 → 计算成功数A

2. 敌人防御检定:
   DP = 敌人防御值
   投骰 → 计算成功数B

3. 对比:
   若 A > B → 命中，伤害 = 基础伤害 + (A - B) × 2
   若 A ≤ B → 未命中
```

### 防御流程（自动）

```
1. 敌人攻击检定:
   DP = 敌人攻击值
   投骰 → 计算成功数C

2. 玩家防御检定:
   DP = 神经反应 + 肉搏技能
   投骰 → 计算成功数D

3. 对比:
   若 D > C → 闪避成功，不受伤害
   若 D ≤ C → 受到伤害 = 敌人伤害 - 防御值
```

### 伤害公式

```
物理伤害:
  基础伤害 = 武器伤害 + 肌肉强度
  技能加成 = 攻击技能等级 × 3
  实际伤害 = 基础伤害 + 技能加成 + (攻击成功数 - 防御成功数) × 2

精神伤害:
  基础伤害 = 技能伤害 + 智力
  技能加成 = 神秘学技能等级 × 3
  实际伤害 = 基础伤害 + 技能加成 + (攻击成功数 - 防御成功数) × 3

最终伤害 = max(1, 实际伤害 - 敌人护甲)
伤害上限 = 基础伤害 × 3

暴击系统:
  暴击条件: 攻击成功数 ≥ 防御成功数 × 2
  暴击效果: 伤害 × 1.5
  暴击伤害上限 = 基础伤害 × 4

多次攻击惩罚:
  同一回合内:
    第1次攻击: 无惩罚
    第2次攻击: -1 DP
    第3次攻击: -2 DP
    第N次攻击: -(N-1) DP
```

## 六、伤害与防御类型

### 伤害类型（3类）

| 类型 | 来源 | 特点 |
|------|------|------|
| **物理伤害** | 近战/远程攻击 | 被护甲减免 |
| **精神伤害** | 法术/技能 | 被魔法抗性减免 |
| **真实伤害** | 暴击/特殊技能 | 无视所有防御 |

### 防御类型（2类）

| 类型 | 效果 | 来源 |
|------|------|------|
| **护甲** | 减免物理伤害 | 免疫强度+装备 |
| **魔法抗性** | 减免精神伤害 | 精神力+装备 |

### 伤害计算流程

```
1. 计算基础伤害
   物理: 武器伤害 + 肌肉强度
   精神: 技能伤害 + 精神力

2. 计算加成伤害
   技能加成 = 技能等级 × 3
   暴击加成 = 暴击时 × 1.5

3. 计算防御
   物理: 伤害 - 护甲
   精神: 伤害 - 魔法抗性

4. 最终伤害
   最终 = max(1, 计算后伤害)
```

### 伤害类型对应关系

```
物理伤害来源:
  肌肉强度 → 物理攻击
  神经反应 → 远程攻击
  肉搏/白刃/枪械/弓术技能 → 物理伤害

精神伤害来源:
  精神力 → 精神伤害
  神秘学技能 → 精神伤害
  智力 → MP资源 (非直接伤害)

防御来源:
  免疫强度 → 护甲
  精神力 → 魔法抗性
```

## 八、基因锁系统

### 核心概念

```
基因锁 = 被动效果 + 主动效果
被动: 解锁后永久生效（属性加成）
主动: 手动开启的额外加成（消耗意志力）
```

### 基因锁解锁机制

| 阶位 | 解锁条件 | 概率计算 |
|------|----------|----------|
| 一阶 | 受到伤害时 | 1% + HP下降% |
| 二阶 | 熟练度 | 1% + 熟练度% |
| 三阶 | 熟练度 | 1% + 熟练度×0.3% |
| 四阶 | 熟练度 | 1% + 熟练度×0.1% |
| 五阶 | 熟练度 | 1% + 熟练度×0.05% |

### 基因锁熟练度

```
熟练度获取:
  每手动开启一次基因锁: 熟练度+1
  每次在副本中激活: 熟练度+2

熟练度效果:
  二阶: 每点+1%解锁概率
  三阶: 每点+0.3%解锁概率
  四阶: 每点+0.1%解锁概率
  五阶: 每点+0.05%解锁概率
```

### 基因锁被动效果（永久）

| 阶位 | 被动效果 | 解锁内容 |
|------|----------|----------|
| 一阶 | 全属性+1 | 基础战斗能力 |
| 二阶 | 全属性+2 | 专长系统 |
| 三阶 | 全属性+3 | 传奇属性解锁 |
| 四阶 | 全属性+4 | 高级战斗特性 |

  5: { allStats: 10, crit: 0.5, dodge: 0.5, cost: 5 }
}

// 基因锁解锁概率计算
function getUnlockChance(
  tier: number,
  当前HP: number,
  maxHP: number,
  proficiency: number
): number {
  let baseChance = 0.01  // 1%
  
  if (tier === 1) {
    // 一阶: 低HP加成
    const hpPercentage = 当前HP / maxHP * 100
    const hpDrop = 100 - hpPercentage
    return Math.min(1, baseChance + hpDrop/100)
  }
  
  // 后续阶段: 熟练度加成
  let proficiencyBonus = 0
  switch(tier) {
    case 2: proficiencyBonus = proficiency * 0.01
      break
    case 3: proficiencyBonus = proficiency * 0.003
      break
    case 4: proficiencyBonus = proficiency * 0.001
      break
    case 5: proficiencyBonus = proficiency * 0.0005
      break
  }
  
  return Math.min(1, baseChance + proficiencyBonus)
}

// 计算离线奖励（基于当前地图）
function calculateOfflineReward(): OfflineReward {
  const now = Date.now()
  const offlineSeconds = (now - player.offlineIdle.lastLogoutTime) / 1000
  
  player.offlineIdle.offlineDuration += offlineSeconds
  
  const availableSeconds = Math.min(
    player.offlineIdle.offlineDuration,
    player.offlineIdle.maxOfflineDuration
  )
  
  const hours = availableSeconds / 3600
  const map = GAME_MAPS.find(m => m.id === player.idle.currentMapId) ?? GAME_MAPS[0]
  const rewardPoints = Math.floor(hours * map.rewardPerHour.rewardPoints)
  const xp = Math.floor(hours * map.rewardPerHour.xp)
  
  return {
    rewardPoints,
    xp,
    duration: availableSeconds
  }
}

// 领取离线奖励
function claimOfflineReward(): void {
  const reward = calculateOfflineReward()
  
  player.currencies.rewardPoints += reward.rewardPoints
  player.experience.current += reward.xp
  player.experience.total += reward.xp
  
  player.offlineIdle.offlineDuration = 0
  player.offlineIdle.lastLogoutTime = Date.now()
}
