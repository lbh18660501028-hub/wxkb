## 一、项目概述

### 核心理念

- **挂机保底**: 自动获取奖励点数（无支线产出）
- **手动副本**: 核心成长路径，获取支线+XP
- **支线互换**: 3:1经典比例，高级线索稀缺
- **三本质系统**: 科技/魔幻/特异三大伤害本质，丰富战斗策略

### 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架（`<script setup>` + Composition API） |
| TypeScript | 类型安全（strict模式） |
| Vite | 构建工具 |
| Pinia | 状态管理（单Store架构） |
| 原生CSS | 赛博朋克主题（scoped样式） |

## 三、核心系统

### 1. 三大货币系统

| 货币 | 挂机获取 | 副本获取 | 用途 |
|------|---------|---------|------|
| 奖励点数 | ✅ 每小时+500 | ✅ 大量 | 购买属性/血统/技能/装备 |
| 支线 (D/C/B/A/S) | ❌ | ✅ 掉落 | 兑换强化能力 |
| XP | ❌ | ✅ 战斗 | 训练技能 |

**挂机收益:**
- 每小时 +500 奖励点数（生化危机地图）
- 每小时 +100 XP（生化危机地图）
- 无支线产出
- 收益随地图等级提升
- 离线上限5小时累积

### 2. 属性系统（九属性）

基于TRPG规则简化，采用三大分组九属性体系：

#### 身体属性（Physical）

| 属性 | 英文 | 定位 | 每点效果 | 附加成功 |
|------|------|------|---------|---------|
| 力量 | strength | 科技输出 | 科技攻击+2 | 6→+1, 11→+2, 16→+3, 21→+4 |
| 敏捷 | agility | 速度/命中 | 闪避+2%, 速度+1, 命中+0.5% | 6→+1, 11→+2, 16→+3, 21→+4 |
| 耐力 | endurance | 生存 | HP上限+10 | 6→+1, 11→+2, 16→+3, 21→+4 |

#### 心智属性（Mental）

| 属性 | 英文 | 定位 | 每点效果 | 附加成功 |
|------|------|------|---------|---------|
| 智力 | intelligence | 魔法资源 | MP上限+10 | 6→+1, 11→+2, 16→+3, 21→+4 |
| 感知 | perception | 暴击/侦察 | 暴击率+1%, 命中+0.5% | 6→+1, 11→+2, 16→+3, 21→+4 |
| 决心 | resolve | 魔幻输出 | 魔幻伤害+3, 意志力+1 | 6→+1, 11→+2, 16→+3, 21→+4 |

#### 社交属性（Social）

| 属性 | 英文 | 定位 | 每点效果 | 附加成功 |
|------|------|------|---------|---------|
| 风度 | presence | 特异输出 | 特异攻击加成 | 6→+1, 11→+2, 16→+3, 21→+4 |
| 操控 | manipulation | 特异输出 | 特异攻击加成 | 6→+1, 11→+2, 16→+3, 21→+4 |
| 沉着 | composure | 防御 | 状态抗性+1%, 减伤+1%, 抗暴+1%, 意志力+1 | 6→+1, 11→+2, 16→+3, 21→+4 |

**属性定位总结：**
- 科技输出: 力量（科技本质攻击）
- 魔幻输出: 决心（魔幻本质攻击）
- 特异输出: 风度 + 操控（特异本质攻击）
- 物理防御: 沉着（减伤+抗性+抗暴）
- 生存能力: 耐力（HP上限）
- 速度闪避: 敏捷（闪避+速度）
- 暴击命中: 感知（暴击率+命中）
- 魔法资源: 智力（MP上限）
- 意志力: 决心 + 沉着

**创建规则：**
- 所有属性初始值 = 1
- 职业加成: 主属性+3, 副属性+2
- 缺陷点数兑换: 1点 → 1天赋点
- 普通人上限 = 5（`ATTRIBUTE_CONFIG.MAX_BASE_ATTRIBUTE`）
- 血统突破后上限提升

**存档迁移：** 旧六属性自动映射 → 力量←肌肉强度, 敏捷←神经反应, 耐力←细胞活力, 智力←智力, 感知←max(反应,智力), 决心←精神力, 风度←精神力, 操控←精神力, 沉着←免疫强度

### 3. 技能系统（10个技能）

技能 = 后天学习的专业能力，**仅作用于副本中的骰子检定**。
技能与战斗系统完全解绑，战斗属性纯由属性+装备驱动。

| 技能 | 关联属性 | 说明 | 职业关联 |
|------|---------|------|----------|
| 运动 | 敏捷 | 攀爬、跳跃、奔跑、游泳 | 刺客/长弓/重炮 |
| 调查 | 感知 | 搜集信息、分析线索、搜索隐藏物品 | 杀手/贪狼 |
| 开锁 | 敏捷 | 开锁、解除陷阱、操作机械装置 | — |
| 黑客 | 智力 | 操作电子设备、电脑、终端 | 大校/贪狼 |
| 医学 | 智力 | 医疗道具使用、生物状态判断 | 军医 |
| 学识 | 智力 | 判断道具物品来历和用途 | 大校/考古 |
| 神秘 | 决心 | 超自然现象感知、神秘力量辨识 | 先知 |
| 动物沟通 | 感知 | 与动物沟通、驯服、安抚 | — |
| 感受 | 感知 | 判断NPC信息、意图、隐藏状态 | 烟鬼/作家 |
| 交际 | 操控 | 说服、交易、求助等对NPC操作 | 队长/游侠 |

**检定公式（基于TRPG核心规则）：**
```
骰池 DP = 关联属性值 + 技能等级
每个D10 ≥ 8 计1成功，掷出10额外加投1个（可连锁）
属性附加成功: ≥6→+1, ≥11→+2, ≥16→+3, ≥21→+4
技能附加成功: 3级→+1, 6级→+2, 9级→+3, 12级→+4
总成功数 ≥ DC 即为检定成功
```

**DC难度表：**
| 难度 | DC | 对应副本 |
|------|-----|---------|
| 简单 | 3-5 | 生化危机 |
| 一般 | 6-8 | 异形巢穴 |
| 困难 | 9-11 | 猛鬼街 |
| 极难 | 12-14 | 寂静岭 |
| 地狱 | 15+ | 死神来了/闪灵 |

**技能成长规则：**
- 职业提供初始技能Lv.1（`startingSkill`）
- 副本技能检定成功 → 有概率自动提升（用进成长）
- XP升级：花费经验值提升技能等级

**技能上限与升级费用：**
- 最大等级: 12级（`SKILL_CONFIG.MAX_LEVEL`）
- 升级费用（XP）: 100/150/200/300/400/500/700/900/1100/1400/1700/2000/2500

## 四、战斗系统（D10骰子 + 三本质）

### 三本质系统

游戏采用三大伤害本质，取代旧版的物理/魔法二分法：

| 本质 | 说明 | 来源 |
|------|------|------|
| **科技本质** (technology) | 借助科学技术构成，可被科学理论解释 | 武器射击、科技装备 |
| **魔幻本质** (fantasy) | 来自魔法、奇幻、灵异体系，脱离现实科学 | 法术、神秘力量 |
| **特异本质** (abnormal) | 科学难以解释，也不属于常规魔法 | 异常力量、特殊能力 |

**旧版兼容：** physical → technology, magical → fantasy（自动映射）

### 核心检定系统

```
DP（骰池）= 属性 + 技能 + 调整值
投骰: DP枚D10
成功: 每枚 ≥ 8 → 1成功
加骰: 投出10 → 再投1枚（可连锁）
附加成功: 高属性/技能额外加成

最终成功数 = 基础成功数 + 附加成功数
```

### 附加成功规则

```
属性附加成功:
  属性 6-10  → +1
  属性 11-15 → +2
  属性 16-20 → +3
  属性 21+   → +4

技能附加成功:
  技能 3-5  → +1
  技能 6-8  → +2
  技能 9-11 → +3
  技能 12+  → +4
```

### 进阶战斗属性（30+项）

战斗系统使用丰富的进阶属性，由属性+装备+血统+基因锁综合计算：

| 属性 | 说明 | 默认值 |
|------|------|--------|
| 科技/魔幻/特异攻击 | 三本质攻击力 | 0 |
| 科技/魔幻/特异防御 | 三本质防御力 | 0 |
| 速度 | 决定先后手 | 0 |
| 暴击率 | 触发暴击概率 | 0 |
| 暴击伤害 | 暴击倍率 | 1.5 |
| 抗暴 | 降低被暴击概率 | 0 |
| 命中 | 攻击命中基础概率 | 0.75 |
| 闪避 | 降低被命中概率 | 0 |
| 反击 | 被攻击后追加反击 | 0 |
| 反伤 | 反弹部分伤害 | 0 |
| 连击 | 追加半额伤害 | 0 |
| 护盾 | 每回合优先抵扣 | 0 |
| 回盾 | 每回合恢复护盾 | 0 |
| 眩晕 | 使目标眩晕概率 | 0 |
| 抗眩 | 降低被眩晕概率 | 0 |
| 吸血 | 按伤害回复生命 | 0 |
| 减伤 | 百分比伤害减免 | 0 |
| 穿透 | 无视固定防御 | 0 |
| 破甲 | 按比例削弱防御 | 0 |
| 格挡 | 降低本次伤害 | 0 |
| 韧性 | 降低暴击伤害 | 0 |
| 真实伤害 | 减伤后追加伤害 | 0 |
| 真实防御 | 抵消真实伤害 | 0 |

### 攻击流程

```
1. 玩家攻击检定:
   DP = 攻击属性 + 技能等级（副本战斗中）
   投骰 → 计算成功数A

2. 敌人防御检定:
   DP = 敌人防御值
   投骰 → 计算成功数B

3. 对比:
   若 A > B → 命中，伤害 = 基础伤害 + (A - B) × 2
   若 A ≤ B → 未命中
```

### 伤害公式

```
基础伤害:
  武器/技能基础伤害 + 属性加成
  技能加成 = 技能等级 × 3
  成功数差值伤害 = (攻击成功数 - 防御成功数) × 2

本质伤害减免:
  科技伤害 → 科技本质防御
  魔幻伤害 → 魔幻本质防御
  特异伤害 → 特异本质防御
  混合伤害 → 三本质各承担1/3

最终伤害 = max(1, 计算后伤害 - 减伤)
伤害上限 = 基础伤害 × 3

暴击系统:
  暴击条件: 攻击成功数 ≥ 防御成功数 × 2
  暴击效果: 伤害 × 1.5
```

## 六、伤害与防御类型

### 伤害类型（4类）

| 类型 | 来源 | 特点 |
|------|------|------|
| **科技本质** | 武器射击/科技装备 | 被科技本质防御减免 |
| **魔幻本质** | 法术/神秘力量 | 被魔幻本质防御减免 |
| **特异本质** | 异常力量/特殊能力 | 被特异本质防御减免 |
| **混合** | 特殊技能 | 三本质各承担1/3 |

### 防御类型（3类）

| 类型 | 效果 | 来源 |
|------|------|------|
| **科技本质防御** | 减免科技伤害 | 装备+耐力+沉着 |
| **魔幻本质防御** | 减免魔幻伤害 | 决心+沉着+装备 |
| **特异本质防御** | 减免特异伤害 | 沉着+装备 |

### 伤害计算流程

```
1. 计算基础伤害
   科技: 武器攻击力 + 力量
   魔幻: 法术基础伤害 + 决心×2
   特异: 技能/特殊能力定义

2. 计算加成伤害
   技能加成 = 技能等级 × 3
   成功数差值 = (攻击成功数 - 防御成功数) × 2
   暴击加成 = 暴击时 × 1.5

3. 计算防御
   对应本质防御减免
   减伤百分比减免
   护盾优先抵扣

4. 最终伤害
   最终 = max(1, 计算后伤害)
```

| fantasy | 使用魔幻本质攻击 | 法杖、魔法武器 |
| abnormal | 使用特异本质攻击 | 异常武器 |

## 八、基因锁系统

### 核心概念

```
基因锁 = 被动效果 + 主动效果
被动: 解锁后永久生效（属性加成）
主动: 手动开启的额外加成（消耗意志力）
```

### 基因锁解锁机制

| 阶位 | 解锁条件 | 概率计算 | 最大熟练度 |
|------|----------|----------|-----------|
| 一阶 | 受到伤害时 | 1% + HP下降% | 10 |
| 二阶 | 一阶熟练度满 | 1% + 熟练度×1% | 18 |
| 三阶 | 二阶熟练度满 | 1% + 熟练度×0.3% | 21 |
| 四阶 | 三阶累计21XP | 1% + 熟练度×0.1% | 36 |
| 五阶 | 四阶熟练度满 | 1% + 熟练度×0.05% | 50 |

### 基因锁熟练度

```
熟练度获取:
  每手动开启一次基因锁: 熟练度+1
  每次在副本中激活: 熟练度+2

熟练度效果:
  二阶: 每点+1%解锁概率
  三阶: 每点+0.3%解锁概率
  四阶: 每点+0.1%解锁概率
  五阶: 每点+0.05%解锁概率
```

### 基因锁被动效果（永久）

| 阶位 | 被动效果 | 解锁内容 |
|------|----------|----------|
| 一阶 | 全属性+1 | 基础战斗能力 |
| 二阶 | 全属性+2 | 细胞级控制 |
| 三阶 | 全属性+3 | 脑域开发 |
| 四阶 | 全属性+4 | 高级战斗特性 |
| 五阶 | 全属性+5 | 终极形态 |

### 基因锁主动效果（手动开启）

| 阶位 | 主动效果 | 激活消耗 |
|------|----------|----------|
| 一阶 | 全属性+2, 暴击+10%, 闪避+10% | 1意志/回合 |

## 十四、统一角色系统

### 核心设计

取消玩家/队友区分，所有角色（包括主角）使用统一的 `Character` 接口。
角色通过 `characters` 数组管理，index 0 为主角。

### Character 接口

```typescript
interface Character {
  id: string                    // 角色唯一ID
  name: string                  // 角色名
  professionId: string          // 职业ID
  gender: string                // 性别
  attributes: Attributes        // 九属性
  equippedItems: Record<string, string>  // 装备槽→物品ID映射
  geneLock: CharacterGeneLock   // 基因锁状态（每角色独立）
  equippedBloodline: string | null       // 已装备血统ID
  skills: Record<string, number>         // 技能ID→等级映射
  spells: Record<string, boolean>        // 法术ID→是否学习
  currentMp: number             // 当前MP
  volumeLevel: number           // 体积等级 0-4
  statusEffects: ActiveStatusEffect[]    // 当前状态效果
}
```

### 基因锁状态（每角色独立）

```typescript
interface CharacterGeneLock {
  tier: number           // 当前阶位 0-5
  proficiency: number[]  // 各阶熟练度
  active: boolean        // 主动效果是否激活
  activeTier: number     // 激活的阶位
}
```

### 队友系统

- 最多招募3名队友（`COMPANION_CONFIG.MAX_COMPANIONS`）
- 招募费用递增: 500/1500/3000 奖励点
- 队友使用与主角相同的 `Character` 接口
- 队友可装备物品、学习技能、解锁基因锁
- 战斗中队友提供自动攻击支援

### 存档迁移

存档格式从旧格式（`name/attributes/geneLock/playerSkills` + `companions[]`）迁移到新格式（`characters[]` 数组），`migrateToCharacters()` 函数处理旧存档兼容。

### 技能检定机制

```
检定公式:
  DP = 属性 + 技能等级 + 调整值
  DC = 节点难度

  投骰: DP枚D10
  成功: 每枚 ≥ 8 → 1成功
  加骰: 投出10 → 再投1枚

  若 成功数 ≥ DC → 检定成功
  若 成功数 < DC → 检定失败
```

## 十九、开发规范

### 架构原则

1. **单Store架构**: 所有游戏状态集中在 `stores/game.ts`
2. **数据与逻辑分离**: `data/` 放静态配置，`config/` 放常量，`systems/` 放算法
3. **数值唯一来源**: `config/combat.ts` 是所有数值常量的唯一来源，禁止硬编码
4. **存档兼容**: 新增字段时要在 `loadSave()` 中提供默认值

### 新增系统流程

1. 在 `types/` 定义类型
2. 在 `data/` 添加数据
3. 在 `config/` 添加配置
4. 在 `stores/game.ts` 实现逻辑
5. 在 `components/pages/` 添加页面

### 编码规范

- TypeScript 严格模式，避免 `any`
- Vue 3 `<script setup lang="ts">` 语法
- Pinia Composition API 风格
- 赛博朋克主题色（CSS变量管理）
- 游戏术语统一中文
