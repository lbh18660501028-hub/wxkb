# 新手副本完整设计方案

# 副本一：《生化危机：深巢实验室》

## 一、副本定位

副本名称：生化危机：深巢实验室
副本类型：新手引导 / 生化感染 / 封闭空间探索 / 图形化走格子 MUD
玩家人数：单人
剧情同伴：随机生成 2 名“新人轮回者 NPC”，固定生成一名 “老关”副本指引NPC
副本时长：30—60 分钟
核心体验：探索、感染、资源分配、机关逃生、NPC 生死选择、最终撤离。

本副本作为玩家第一次进入主神副本的教学关，主要让玩家熟悉以下系统：

1. 点击格子移动；
2. 点击房间动作按钮进行搜索、开门、修复、黑客、破坏、取样；
3. 使用背包道具；
4. 与 NPC 对话、分配资源、决定是否同行；
5. 触发强制战斗、随机战斗、操作触发战斗；
6. 理解感染值、警戒值、自毁倒计时；
7. 完成主线和支线目标；
8. 获得临时道具和永久奖励。

---

## 二、世界观与剧情开场

玩家第一次进入主神空间后，还没有解锁“第二角色创建权限”，因此只能以单人身份进入第一个副本。

副本开场：

玩家在一节停运的地下列车中醒来，耳边传来冰冷提示：

“新人轮回者已投放。
当前副本：深巢实验室。
主线任务：在实验室自毁前逃离。
警告：本副本存在感染机制，感染值达到 100 后将死亡或转化。”

玩家并非唯一新人。系统同时投放了数名普通人，他们也能听见主神提示。
这些人被称为“新人轮回者 NPC”。他们看上去像玩家一样被卷入副本，但本质上是剧情 NPC。

## 三、现代图形化交互设计

本游戏底层是 MUD，但界面不使用命令行，而是采用现代 UI。


#### A. 顶部状态栏

显示：

* 感染值；
* 警戒值；
* 当前回合数；
*自毁倒计时

示例：

/ 感染 15 / 警戒 35 / 回合 24


---

#### B. 中央地图区

显示格子地图。

每个格子用不同状态表现：

| 格子状态  | 表现     |
| ----- | ------ |
| 未探索   | 黑色或迷雾  |
| 已探索   | 正常显示   |
| 当前所在  | 高亮边框   |
| 可移动   | 发光边框   |
| 锁定房间  | 红色锁图标  |
| 有战斗   | 红色危险图标 |
| 有剧情   | 黄色感叹号  |
| 有可搜索物 | 放大镜图标  |
| 有 NPC | 人物头像图标 |
| 主线目标  | 蓝色目标图标 |
| 支线目标  | 紫色目标图标 |

玩家点击相邻可移动格子即可移动。
移动消耗 1 回合。

---

#### C. 房间信息卡

玩家进入房间后，显示房间描述。

包括：

* 房间名称；
* 环境描述；
* 危险提示；
* 可见物品；
* 可见 NPC；
* 当前房间可执行操作。

示例：

房间：医疗室
描述：刺鼻的消毒水味混着血腥味，手术台上还亮着冷白色的灯。角落里有一个穿白大褂的女人，她正用手压着肩膀上的伤口。

可见对象：

* 秦岚，医疗研究员；
* 医疗柜；
* 手术台；
* 血迹。

---

#### D. 动作按钮区

不使用输入指令，全部改成按钮。

基础按钮：

| 按钮   | 功能           |
| ---- | ------------ |
| 移动   | 点击地图格子完成     |
| 搜索   | 搜索当前房间       |
| 检查   | 检查特定对象       |
| 开门   | 尝试打开门        |
| 使用道具 | 从背包选择道具使用    |
| 交谈   | 与 NPC 对话     |
| 黑客   | 对终端、门禁、摄像头使用 |
| 修复   | 对发电机、电梯、阀门使用 |
| 破坏   | 破坏培养罐、门锁、摄像头 |
| 取样   | 对样本柜、病毒箱使用   |
| 休息   | 仅安全屋可用       |
| 撤离   | 到达撤离点后显示     |

房间特定按钮会根据配置动态出现。

例如：

在发电机房显示：

* 检查发电机；
* 插入备用保险丝；
* 强行启动；
* 返回。

在医疗室显示：

* 搜索医疗柜；
* 使用手术台；
* 与秦岚交谈；
* 给秦岚抗病毒喷雾。

在激光走廊入口显示：

* 观察激光轨迹；
* 强行通过；
* 使用激光频率记录；
* 返回监控室。

---

#### E. 后勤保障部

渗透载荷区显示玩家拥有的道具。

道具分为：

* 临时道具；
* 永久道具；
* 任务道具；
* 医疗道具；
* 武器道具；
* 消耗品。

玩家点击道具后出现可用目标。

例如点击“抗病毒喷雾”后显示：

* 对自己使用；
* 对秦岚使用；
* 对XXNPC使用；
* 取消。

NPC 卡片显示：

* 名字；
* 身份；
* 当前生命状态；
* 感染状态；
* 信任值；
* 恐惧值；
* 是否跟随；
* 可交互按钮。

NPC 卡片按钮：

| 按钮   | 功能              |
| ---- | --------------- |
| 交谈   | 打开对话            |
| 邀请同行 | 请求 NPC 跟随       |
| 让其留守 | NPC 停留当前房间      |
| 给予道具 | 从背包分配道具         |
| 请求协助 | 搜索、修复、黑客、战斗支援   |
| 查看状态 | 查看 NPC 生命、感染、情绪 |
| 放弃   | 不再保护该 NPC       |

## 四、回合机制

虽然界面现代，但底层仍然是回合式 MUD。

以下行为消耗回合：

| 行为         | 消耗     |
| ---------- | ------ |
| 移动到相邻格子    | 1 回合   |
| 普通搜索       | 1 回合   |
| 彻底搜索       | 2 回合   |
| 黑客         | 1 回合   |
| 修复设备       | 1—2 回合 |
| 战斗一轮       | 1 回合   |
| 与 NPC 关键对话 | 1 回合   |
| 治疗         | 1 回合   |
| 破坏设备       | 1 回合   |
| 取样         | 1 回合   |

不消耗回合：

* 查看地图；
* 查看背包；
* 查看任务；
* 查看 NPC 状态；
* 查看已解锁资料；
* 调整装备。

## 五、搜索模式

每个房间可以提供两种搜索按钮：（每个房间只能搜索1次，避免重复搜索）

### 1. 普通搜索

按钮文字：快速搜索
消耗：1 回合
特点：安全，收益较低。

可能获得：

* 少量弹药；
* 绷带；
* 房间线索；
* 普通钥匙。

---

### 2. 深度搜索

按钮文字：彻底搜索
消耗：2 回合
特点：收益更高，但可能触发危险。

可能结果：

| 结果     | 说明       |
| ------ | -------- |
| 找到隐藏道具 | 获得稀有临时道具 |
| 找到剧情线索 | 解锁支线     |
| 触发尸变   | 操作触发战斗   |
| 触发警报   | 警戒值上升    |
| 感染污染   | 感染值上升    |
| 无发现    | 白白浪费时间   |

## 六、核心系统

### 1. 感染值

玩家和 NPC 都有感染值。

范围：0—100。

| 感染值   | 状态    |
| ----- | ----- |
| 0—29  | 正常    |
| 30—59 | 轻度感染  |
| 60—79 | 中度感染  |
| 80—99 | 重度感染  |
| 100   | 死亡或转化 |

感染来源：

| 来源        | 感染增加 |
| --------- | ---- |
| 被普通感染者抓伤  | +5   |
| 被感染者咬伤    | +10  |
| 被感染犬攻击    | +12  |
| 接触污染血池    | +5   |
| 打开污染冷冻柜失败 | +8   |
| 被变异体重击    | +15  |
| 触发病毒喷雾陷阱  | +20  |

降低感染：

| 方式     | 效果           |
| ------ | ------------ |
| 抗病毒喷雾  | 感染 -15       |
| 抗病毒血清  | 感染 -40       |
| 医疗室消毒  | 感染 -10       |
| 完成抗体支线 | 后续感染增长降低 30% |

---

### 2. 警戒值

全局变量：security_alert。

范围：0—100。

| 警戒值   | 状态        |
| ----- | --------- |
| 0—29  | 低警戒       |
| 30—59 | 摄像头苏醒     |
| 60—79 | AI 开始定位玩家 |
| 80—99 | 精英敌人巡逻    |
| 100   | 清除协议启动    |

警戒值增加：

| 行为       | 增加  |
| -------- | --- |
| 开枪       | +3  |
| 爆破开门     | +15 |
| 黑客失败     | +10 |
| 破坏摄像头    | +5  |
| 拿走病毒样本   | +15 |
| 攻击 AI 核心 | +30 |

警戒值影响：

* 提高随机遭遇概率；
* 提高安全无人机出现概率；
* 提高最终撤离战难度；
* 高警戒时部分 NPC 恐惧值增加。

---

### 3. 自毁倒计时

进入 AI 主机房并处理 AI 后开启。

默认倒计时：50 回合。

开启后：

* 地图部分通道封锁；
* 撤离路线开放；
* 随机遭遇概率上升；
* 感染体移动速度提高；
* 最终 BOSS 开始追踪玩家；
* 玩家必须前往撤离列车。

## 七、随机新人轮回者 NPC 系统

由于玩家暂时无法创建第二角色，所以副本开局会生成 3 名新人轮回者 NPC。（生化危机副本为2个新人+一个引导轮回者，引导轮回者是生化危机副本特有）

他们不是假装在线玩家，而是剧情内和玩家一样被主神投放的 NPC。

### 1. NPC 数量

每局生成 3 人。

固定类型：

| 类型    | 功能        |
| ----- | --------- |
| 冷静分析型 | 教学、解谜、提示  |
| 冲动莽撞型 | 战斗辅助、危险演示 |
| 自私求生型 | 资源冲突、背叛可能 |

玩家最多只能长期带 1 名 NPC 同行。
其他 NPC 会在剧情节点分散、受伤、背叛、感染或死亡。

---

### 2. NPC 属性

每名 NPC 拥有：

| 属性           | 说明    |
| ------------ | ----- |
| hp           | 生命值   |
| infection    | 感染值   |
| trust        | 对玩家信任 |
| fear         | 恐惧值   |
| role         | 类型    |
| skill        | 特长    |
| follow_state | 是否跟随  |
| fate_flag    | 命运标记  |
| hidden_trait | 隐藏倾向  |

---

### 3. NPC 信任值

范围：-100 到 100。

| 信任值        | 表现         |
| ---------- | ---------- |
| -100 到 -50 | 敌视，可能背叛    |
| -49 到 0    | 不信任，拒绝协助   |
| 1 到 49     | 普通合作       |
| 50 到 79    | 主动提醒危险     |
| 80 到 100   | 愿意牺牲或冒险救玩家 |

信任变化：

| 玩家行为       | 变化   |
| ---------- | ---- |
| 给医疗道具      | +15  |
| 救 NPC 脱险   | +25  |
| 抛弃 NPC     | -40  |
| 抢 NPC 道具   | -30  |
| 攻击 NPC     | -100 |
| 带 NPC 完成支线 | +20  |
| 隐瞒感染风险     | -20  |

---

### 4. NPC 恐惧值

范围：0 到 100。

| 恐惧值   | 表现             |
| ----- | -------------- |
| 0—29  | 稳定             |
| 30—59 | 紧张             |
| 60—79 | 容易失控           |
| 80—99 | 可能逃跑、尖叫、误触机关   |
| 100   | 崩溃，脱离队伍或做出极端行为 |

恐惧上升：

* 看到尸体；
* 看到 NPC 死亡；
* 被感染体围攻；
* 玩家低生命；
* 倒计时开启；
* 警戒值过高；
* 玩家对 NPC 施压。

恐惧降低：

* 玩家安慰；
* 玩家分配武器；
* 完成安全屋休整；
* 秦岚治疗；
* 成功击败强敌。

## 十、房间配置总表

### B3：地铁站

类型：出生点。
首次事件：玩家醒来，看到另外 3 名新人轮回者。
可见对象：尸体、破损通讯器、铁轨、车门。
按钮：

* 检查尸体；
* 检查通讯器；
* 与新人交谈；
* 前往列车。

可获得：

* 保安钥匙；
* 破损通讯器线索。

战斗：无。

---

### C3：停运列车

类型：探索。
按钮：

* 搜索座椅；
* 检查驾驶室；
* 检查车厢血迹；
* 返回地铁站；
* 前往月台。

可获得：

* 手电筒；
* 破损地图；
* 少量弹药。

战斗：无。

---

### D3：装卸月台

类型：新手强制战斗。
首次进入触发感染员工 ×2。
按钮：

* 攻击；
* 防御；
* 使用道具；
* 请求 NPC 掩护；
* 逃向列车。

战斗胜利后：

* 解锁战斗教程；
* 获得蓝色权限卡；
* NPC 信任根据表现变化。

---

### E3：安检大厅

类型：门禁分歧。
按钮：

* 使用蓝色权限卡；
* 尝试黑客门禁；
* 检查安检机；
* 讨论路线；
* 强行破门。

结果：

* 使用蓝色权限卡：安全进入；
* 黑客失败：警戒值 +10；
* 强行破门：警戒值 +15；
* 讨论路线：触发新人第一次分歧。

---

### D1：配电室

类型：主线道具房。
按钮：

* 搜索配电柜；
* 取出备用保险丝；
* 检查烧焦痕迹；
* 关闭漏电开关。

取得备用保险丝后触发感染鼠群。
战斗：操作触发。

---

### C2：发电机房

类型：主线修复房。
按钮：

* 检查发电机；
* 插入备用保险丝；
* 强行启动；
* 请求 NPC 协助；
* 返回维修走廊。

结果：

* 正确修复：power_restored=true；
* 强行启动：玩家受伤，警戒值 +5；

## 十一、战斗系统

显示：

* 敌人列表；
* NPC 同伴状态；

* 敌人行动预告。

### 玩家按钮

| 按钮        | 功能            |
| --------- | ------------- |

# 七、E3 第一次路线分歧

原设计写“讨论路线”，但没有内容。V2 明确为一个多选弹窗事件。

触发地点：

E3 安检大厅。

触发条件：

首次到达 E3，且 3 名新人轮回者至少有 2 人存活。

事件名：

**新人第一次分歧**

剧情：

安检大厅的主门被锁住，监控摄像头缓慢转向众人。新人们开始争吵：有人想一起行动，有人想分头找钥匙，有人想直接砸门。

玩家选项：

| 选项           | 效果                                    |
| ------------ | ------------------------------------- |
| 说服所有人一起行动    | 冷静型信任 +10，自私型恐惧 +5；安全但探索效率低           |
| 分成两组寻找钥匙     | 冲动型信任 +10；随机一名 NPC 可能受伤；额外获得补给概率 +20% |
| 让冷静型新人协助破解门禁 | 若冷静型存活，黑客成功率 +20%；失败警戒 +10            |
| 命令冲动型新人破门    | 冲动型信任 +5；警戒值 +15；可能提前触发感染者            |
| 玩家独自行动       | 全体新人信任 -10，自私型可能偷走一个小道具               |
| 强行压服所有人      | 短期恐惧 -5，但全体信任 -20；后续背叛概率上升            |

推荐默认：

第一次游玩建议引导玩家选择“让冷静型新人协助破解门禁”，这样可以展示 NPC 功能，而不是纯剧情摆设。

# 八、K3 红后说服选项前置条件

原设计写“说服 Red Queen 需要公司机密档案”，但 K3 配置没有标注。V2 改为明确按钮状态。

## 1. 按钮显示规则

K3 红后主机房显示以下按钮：

| 按钮        | 显示条件                 |
| --------- | -------------------- |
| 关闭红后      | 默认显示                 |
| 尝试说服红后    | 默认显示，但条件不足时标注“成功率极低” |
| 上传保护伞黑幕证据 | 拥有公司机密档案时显示          |
| 复制红后子程序   | 拥有数据盘时显示             |
| 攻击红后核心    | 默认显示                 |
| 查看主机日志    | 默认显示                 |

## 2. 说服成功条件

说服 Red Queen 的基础成功率为 0。

加成：

| 条件                 | 成功率  |
| ------------------ | ---- |
| 拥有公司机密档案           | +50% |
| 拥有 Lisa Addison 资料 | +20% |
| Kaplan 存活并协助黑客     | +20% |
| Matt Addison 存活    | +10% |
| 玩家此前没有破坏摄像头        | +10% |
| 警戒值低于 50           | +10% |

最高成功率封顶：90%。

## 3. 条件不足时点击“尝试说服”

如果玩家没有公司机密档案，仍然可以点击，但结果大概率失败。

失败结果：

* Red Queen 拒绝；
* 警戒值 +15；
* 刷新安全无人机 ×1；
* “说服红后”按钮锁定；
* 玩家仍可选择关闭、复制或攻击。

失败对白：

“你没有证据。你只是另一个污染源。”

# 九、I4 与 F5 的 BOSS 削弱机制

原设计里 I4 和 F5 都能削弱 BOSS，但没有上限。V2 明确如下。

最终 BOSS：

**舔食者 Licker**

基础生命值：180。

削弱来源：

| 来源         | 数量  | 每次效果             | 最高效果 |
| ---------- | --- | ---------------- | ---- |
| I4 生物罐区控制阀 | 1 个 | Licker 最大生命 -10% | -10% |
| F5 培养室培养罐  | 3 个 | 每个 -10%          | -30% |
| 液氮罐战前冷冻    | 1 次 | Licker 第一回合无法行动  | 不减生命 |

永久削弱上限：

**最多削弱 40% 最大生命。**

也就是：

| 削弱层数 | BOSS 最大生命 |
| ---- | --------- |
| 0 层  | 180       |
| 1 层  | 162       |
| 2 层  | 144       |
| 3 层  | 126       |
| 4 层  | 108       |

实现变量：

```json
{
  "boss_weakened_level": 0,
  "boss_weakened_max": 4,
  "i4_valve_destroyed": false,
  "f5_tank_destroyed_count": 0,
  "liquid_nitrogen_used_on_boss": false
}
```

I4 只有 1 个控制阀。
F5 有 3 个培养罐。
合计最多 4 层削弱。

# 十、F4 冷藏样本路线依赖

原设计主线先到 F4 样本库，但冷藏样本需要 G4 液氮罐，确实有路线依赖没有说明。V2 明确为分支路线。

## F4 样本库首次到达时

玩家看到 T 病毒样本柜。

按钮：

| 按钮           | 条件    | 结果                          |
| ------------ | ----- | --------------------------- |
| 直接取走 T 病毒样本箱 | 默认    | 获得“不稳定 T 病毒样本箱”，警戒 +15      |
| 销毁样本         | 默认    | 进入净化路线                      |
| 暂不取样，标记样本柜   | 默认    | 设置 sample_marked=true，可稍后返回 |
| 冷藏取样         | 需要液氮罐 | 获得“冷藏 T 病毒样本”               |
| 检查破碎柜门       | 默认    | 获得 Licker 伏笔                |

## 正确冷藏路线

推荐路线：

F4 样本库 → 选择“暂不取样，标记样本柜” → G4 冷冻室 → 取得液氮罐 → 返回 F4 → 冷藏取样。

## 如果玩家先直接取走样本

获得：

**不稳定 T 病毒样本箱**

效果：

* 主线可以继续；
* 结算奖励低于冷藏样本；
* 自毁倒计时后每 10 回合有 10% 概率导致警戒值 +5；
* 若玩家感染值 ≥ 80，最终撤离时可能触发“样本污染”判定。

## 如果玩家冷藏取样

获得：

**冷藏 T 病毒样本**

效果：

* 结算奖励提高；
* 可进入黑箱路线；
* 不触发样本污染判定；
* 可解锁后续研究路线。

| ------- | ----------------------------------- |

| 区域      | 变化                         |

# 十二、结局 D 判定修正

原设计“倒计时最后 5 回合内撤离”容易误解。V2 明确为：

当玩家点击 K5 的“完成撤离”按钮时，如果：

```json
countdown_remaining >= 1 && countdown_remaining <= 5
```

则满足“最后 5 回合内撤离”。

不是指副本总第 46—50 回合，而是指自毁倒计时剩余 1—5 回合。

结局 D 条件：

| 条件                           | 必须满足 |
| ---------------------------- | ---- |
| 玩家成功撤离                       | 是    |
| countdown_remaining 在 1—5 之间 | 是    |
| 玩家感染值 ≥ 80 或至少 2 名 NPC 死亡    | 满足其一 |
| 未达成黑箱路线或净化路线                 | 是    |

结局名称：

**惨胜：最后一班列车**
