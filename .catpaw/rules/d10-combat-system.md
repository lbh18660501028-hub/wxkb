# D10 骰子战斗系统规则

本规则在讨论战斗系统、骰子检定、伤害计算等内容时生效。

## D10 检定核心规则（来源：设定集/核心规则TRPG版.md）

### 1. DP（骰池）
```
DP = 属性值 + 技能值 + 调整值
```
DP决定投掷D10的数量。特殊情况下DP可能等于单属性、双属性之和或定值。

### 2. 成功判定
- 投出 **8, 9, 10** → 1个成功
- 投出 **1-7** → 无成功
- 成功数 = 所有8/9/10的骰子总数

### 3. 加骰（Explosion）
- 投出 **10** → 再投1枚D10
- 新骰子同样遵循 8/9/10 = 成功
- 可连锁：10 → 10 → 10...
- 部分能力提供"9加骰"（9或10都加骰），最低加骰值为8

### 4. 附加成功
- 属性值 ≥6 → +1附加成功
- 属性值 ≥11 → +2附加成功
- 属性值 ≥16 → +3附加成功
- 属性值 ≥21 → +4附加成功
- **关键规则：附加成功不能使失败变成功**（投骰成功数为0时附加成功无效）

### 5. 难度对比
- **竞争**：DC是固定预设值（如举重、跳跃）
- **对抗**：DC是对手的检定成功数（如潜行vs侦查、攻击vs防御）

### 6. 机运骰
- DP ≤ 0 时仍投1枚D10
- 仅10 = 成功，1 = 大失败
- 不受增加DP或检定结果的能力影响

## 战斗结算流程

### 攻击检定
1. 攻击DP = 攻击属性 + 武器技能 + 调整值
2. 投骰 → 计算成功数 + 附加成功 = 总成功数

### 防御检定
1. 防御DP = 防御属性 + 防御技能 + 调整值
2. 投骰 → 计算成功数 + 附加成功 = 总成功数

### 伤害计算
```
有效成功数 = 攻击总成功数 - 防御总成功数
基础伤害 = COMBAT_CONFIG.BASE_DAMAGE (10)
伤害 = 基础伤害 + 有效成功数 × DIFF_DAMAGE_MULTIPLIER (2)
最终伤害 = max(伤害 - 护甲减免, MIN_DAMAGE)
```

### 暴击
- 触发条件：攻击成功数 ≥ 防御方防御值 × CRIT_SUCCESS_RATIO (2)
- 暴击伤害 = 正常伤害 × CRIT_MULTIPLIER (1.5)

### 伤害类型
- **technology（科技本质）**：受科技本质防御减免
- **fantasy（魔幻本质）**：受魔幻本质防御减免
- **abnormal（特异本质）**：受特异本质防御减免
- **mixed（混合）**：三本质各承担1/3
- 旧字段 physical → technology，magical → fantasy
